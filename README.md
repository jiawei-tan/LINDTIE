<div align="center">
	<h1>LINDTIE</h1>
	<p>
		<b>Identify aberrant transcripts in cancer using long-read RNA-seq</b>
	</p>
</div>

LINDTIE is a Nextflow-based pipeline for detecting aberrant cancer transcripts from long-read RNA-seq data, extending beyond canonical fusions to identify a full range of transcriptome rearrangements. It uses a reference-free workflow — combining de novo assembly, quantification, differential expression, and annotation — to discover novel variants from raw case–control FASTQ/FASTA files.

For detailed usage instructions and guides, check out the [LINDTIE documentation](https://jiawei-tan.github.io/LINDTIE/).
