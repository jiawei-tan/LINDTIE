Quick Start Guide

Get up and running with LINDTIE in just a few minutes. This guide covers the essential steps to begin your bioinformatics analysis.

i

Prerequisites

Before beginning, ensure you have Python 3.8+ and the required dependencies installed.

Installation

Install LINDTIE using pip or conda: 

# Using pip
pip install lindtie

# Using conda
conda install -c bioconda lindtie

Basic Usage

Here's a simple example to process RNA-seq data: 

import lindtie as lt

# Load your data
data = lt.load_fastq("sample.fastq")

# Process sequences
results = lt.analyze(data, method="quality_control")

# Export results
results.to_csv("output.csv")
